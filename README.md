# typescript_grpc

Project demo với 4 loại kết nối:
- Unary
- Server Streaming
- Client Streaming
- Bidirectional Streaming
## Project setup
`yarn install`

## Runs the gRPC server
`yarn start`

## Runs out client
`yarn client`

## Runs out client chat 1
`yarn client <username1>`

## Runs out client chat 2
`yarn client <username2>`
